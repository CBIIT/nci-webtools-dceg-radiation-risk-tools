Guidance for Entering Distributions in IREP

This page provides guidance for entering distribution parameters for dose probability distributions. Depending on the type of distribution selected, the user is required to enter either 1, 2 or 3 parameters to describe the distribution. In cases where less than 3 parameters are required, any values entered in fields for the other parameters are ignored.
