Upload Saved File

The NCI Radiation Risk Assessment Tool has been designed to accept a preformatted input file. When uploaded into RadRAT, the input file will populate all input screens and set all system variables. Download a template version of the Excel input file (filled with default values) and save it on your computer.

Then

  • Click the "Browse" button to locate the input file on your computer.
  • After the path to the file is shown on this screen, click "Upload File" to complete the process.

[Note: The file must be of the proper template format to be usable.]
