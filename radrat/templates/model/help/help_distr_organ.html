Guidance for Entering Organ Doses

For a dose with negligible uncertainty or to ignore uncertainty in dose, select "Fixed Value" as the Distribution Type and enter the dose value in the Parameter 1 column.

Uncertainty in dose may be described using probability distributions. To enter an uncertain dose, select one of the distributions from the Distribution Type menu and enter the parameters required to define the selected distribution. The parameters required to define each distribution type are provided below.

In cases where fewer than 3 parameters are required, unnecessary parameters are inactive.
